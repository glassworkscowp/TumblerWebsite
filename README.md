# Glassworks Co Website

A static website for Glassworks Co, a student-owned business selling sustainable glass tumblers.

## Deployment to GitHub Pages

1. Create a new repository on GitHub.
2. Upload all files from this project to the repository.
3. Go to the repository settings.
4. Scroll down to "Pages" section.
5. Under "Source", select "Deploy from a branch".
6. Select the main branch and save.
7. The site will be available at `https://yourusername.github.io/repository-name/`.

## Features

- Home page with product showcase
- About Us page
- Team page
- Catalogue with add to cart functionality
- Order page with Google Form link

## Technologies Used

- HTML5
- CSS3
- JavaScript (vanilla)
